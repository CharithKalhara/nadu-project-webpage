Court Case Management System

A web-based case management application built with PHP and PostgreSQL, containerized using Docker, and deployed on AWS with secure user authentication and real-time case tracking.

Key Features

Case Record Management

Create, update, view, and delete court case records through a user-friendly web interface.

Advanced Search & Filtering

Quickly search and filter case records by year, reference number, company name, received date, and case status.

Secure User Authentication

Protects user access with Supabase Authentication and session-based authorization controls.

PostgreSQL Database Integration

Stores and manages case information securely using PostgreSQL with PDO database connectivity.

Real-Time System Monitoring

Displays database status, response time, server latency, and live system information for administrators.

Docker-Based Deployment

Supports containerized deployment using Docker and Docker Compose for simplified setup and maintenance.

About the Project

Court Case Management System is a PHP and PostgreSQL-based web application designed to manage court case records efficiently. The system provides secure authentication using Supabase, supports CRUD operations, advanced search functionality, and is deployed using Docker containers on a Linux server.
